Mi Primer Proyecto con Django

Este es mi primer proyecto desarrollado con Django, un blog sencillo en el que los artículos se administran y publican desde el panel de administración de Django. Está construido utilizando únicamente Django, HTML y CSS, con el objetivo de practicar y afianzar mis conocimientos en el desarrollo web.

A través de este proyecto he aprendido:

Cómo estructurar una aplicación en Django

Cómo crear modelos y gestionarlos desde el admin

Cómo renderizar plantillas HTML

Cómo aplicar estilos con CSS

Cómo mostrar contenido dinámico en la interfaz

Este blog es una base para seguir aprendiendo y construyendo proyectos más completos con Django.
